Контрольні питання та відповіді

  1. Що таке CSS і яка його роль у веб-розробці?

    CSS (Cascading Style Sheets) — це мова стилів, яка використовується для опису зовнішнього вигляду HTML-документів. Вона дозволяє задавати кольори, шрифти, відступи та інші стилі, розділяючи структуру і дизайн.

  2. Які основні властивості в CSS і як вони задаються?

    Основні властивості включають фон (background-color), текст (color, font-size), відступи (margin, padding), межі (border) та розміри (width, height). Задаються через селектори:

    p {
    color: red;
    font-size: 16px;
}
  3. Що таке селектори в CSS? Наведіть приклади різних типів селекторів.

    Селектори — це способи вибору HTML-елементів для застосування стилів. Приклади:

    • За тегом: p { color: green; }
    • За класом: .class { font-size: 14px; }
    • За ID: #id { text-align: center; }
  4. Яка різниця між класовим та ідентифікаторним селекторами?

    Класовий селектор застосовується до багатьох елементів і починається з крапки (.class), а ідентифікаторний селектор унікальний для одного елемента і починається з решітки (#id).

  5. Як записати коментарі в CSS?

    Коментарі записуються між /* і */, наприклад:

    /* Це коментар */
  6. Як змінити фоновий колір, колір тексту та розмір шрифту для елементу?

    Приклад:

    div {
    background-color: lightblue;
    color: white;
    font-size: 16px;
}
  7. Як задати межі (borders), відступи (margins) та внутрішні відступи (padding) для елементів в CSS?

    Приклад:

    div {
    border: 2px solid black;
    margin: 10px;
    padding: 5px;
}
